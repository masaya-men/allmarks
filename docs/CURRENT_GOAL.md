- **1 機能ずつ、フレッシュなセッションで**進める（文脈が軽い＝Claude が速く・安く・正確に走れる）。

  > 「docs/CURRENT_GOAL.md を読んで、バックログの **#N** を進めて。設計は最小往復で最速、実装 → 検証 → デプロイまで自走して。」

## 直近の完了（s173 この会話）＝ #2 TUNE 刷新セット **全完了・本番反映済**

- **#1 角丸 ON/OFF トグルを TUNE に同居**（`CORNERS ROUND/SQUARE`）：TUNE ドロワー左列（プリセットの下・破線区切り）に mixer 風トグル。OFF で全カード角ゼロ（ライトボックスも追従）、ON で従来の size-aware 角丸。`BoardConfig.roundedCorners`（既定 true）で永続。CardsLayer の per-card `--card-radius` に配線。
- **#4 カード幅/ギャップのスナップ**：`N×幅 + (N−1)×ギャップ = 盤面幅` になる「左右余白が揃う」値を純関数 `lib/board/fill-snap.ts` で算出。W/G フェーダーに**緑の目印**を表示、**離した瞬間**その近く（±10px）なら吸着。ドラッグ中は自由・**Shift 微調整中は無効**。
- 検証：tsc0 / **vitest 2097/0** / クリーンビルド / e2e 3本（角丸トグル切替・リロード永続・目印描画）実測 PASS。Playwright スクショで丸⇔四角＋緑目印を目視。
- **同セッションでユーザー実機フィードバック2件も修正・本番反映済**：
  - **①角丸OFFなのにライトボックス復帰で角が丸まる** → radius ミラーの削除タイミングがモーフ途中だったのを「完全終了後のみ削除」に分離して根治。
  - **②再生中の動画クリック→移行時に一瞬音あり** → モーフのクローン `<video>` が cloneNode で muted プロパティを失い音付き autoplay していたのを、クローンを muted 化して解消。
- **ユーザー実機目視の残**：CORNERS 操作感／スナップ吸着（±10px）／緑目印の視認性（フェーダー吸着は Playwright 駆動不可＝実機のみ）／**①ライトボックス復帰で角が四角のままか**／**②動画クリックで移行時に無音か**（ライトボックスは Playwright で開けず自動検証不可＝実機のみ）。
